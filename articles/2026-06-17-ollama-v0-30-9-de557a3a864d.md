# Ollama v0.30.9

New Ollama release with Cohere2Moe architecture support and critical bug fix for local coding agents. Passes Ships Test.

<h2>What's Changed</h2>
<ul>
<li>Support for Cohere2Moe architecture</li>
<li>Fixed LFM2 parser/render for cases where thinking was not emitted</li>
<li>Fixed issue where <code>ollama launch claude</code> and other coding agent or assistant use cases would only output one token</li>
<li>Ollama will now return an error if a single message is larger than the current context window</li>
</ul>
